# Advent of Code 2025

This repository contains my solutions for **Advent of Code 2025**, implemented in Python.

Each day lives in its own script (for example, `day-01.py`) and typically reads its input from the corresponding file(s) under the `input/` directory.

## Running a Day

Most days can be run from the command line using Python 3:

```bash
python day-01.py --input input/day-01/part-one
```

Adjust the day number and input path as needed.
